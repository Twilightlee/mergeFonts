# mergeFonts
merge fonts by fontforge

## Requirement: intall fontforge

### linux : 
               sudo apt install fontforge

### macOS: 
               brew install fontforge

### Windowns: 
               download install file from offical web site

---

## usage: put fonts file and merge_fonts.py in same directory. 

**fontforge -lang=py -script merge_fonts.py**

---

** Note: default fonts in merge_fonts are JetBrainsMonoNerdFontMono* and LXGWWenKaiMono*. **

**Change fonts as you want.**
